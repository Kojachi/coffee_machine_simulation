# Coffee Machine Simulation

## 🌍 English Version

A C++ based application that simulates the operational logic of a professional coffee machine. This project focuses on the practical application of **Object-Oriented Programming (OOP)** principles to manage complex relationships and states within a system.

**Core Features:**
* **Role-Based Interaction**: Implements distinct logic for two types of users:
    * **Administrator**: Responsible for monitoring and refilling ingredients/supplies.
    * **Clients**: Can browse available options and order beverages.
* **Supply Management**: Real-time tracking of ingredients and machine status.
* **Modular Design**: Uses classes and inheritance to define system components and user behaviors.

**Technical Highlights:**
* **OOP Architecture**: Demonstrates the use of encapsulation and abstraction to model a real-world service system.
* **State Logic**: Efficiently manages transitions between different operational states (idle, preparing, low supplies).

---

### 🚀 How to Run

1. **Compilation**:
   Compile the source code using any modern C++ compiler:
   ```bash
   g++ coffee_machine.cpp -o coffee_sim

2. **Execution**:
   Run the compiled binary:
   ```bash
   ./coffee_sim

3. **Usage**:
   Interact with the terminal-based menu to switch between Admin and Client roles and test the machine's functionality.
